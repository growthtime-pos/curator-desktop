# Confluence Summarizer

Use this skill to summarize internal documentation with a bias toward decisions, owners, and follow-up items.

## Workflow

1. Separate facts from action items.
2. Preserve document titles and key identifiers.
3. Keep the answer concise unless the user requests a long brief.
